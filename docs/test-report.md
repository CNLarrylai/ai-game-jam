## 核心验证点

### ✅ 评论→剧情生成链路
- 观众评论成功被AI融入叙事（@用户名被引用）
- 评论归属(attribution)正确返回
- 创造性评论（"生成一个泡面工厂"）→ 生成全新地点选项
- 噪音评论（"666""加油"）不会触发事件

### ✅ 数值反向逻辑
- 喝水 → thirst:-15 ✅（口渴减少=好事）
- 吃罐头 → hunger:-20 ✅（饥饿减少=好事）
- 鲱鱼难吃 → sanity:-8 ✅（精神下降=符合设定）
- resource_adjust场景 → resourceChanges直接返回数值 ✅

### ✅ 黑色幽默基调
- "葱油饼味道的AI陷阱"
- "光头厨师梦到红烧肉，醒来只有你"
- "橘猫精神状态比你还稳定"
- "马斯克因为对某个AI说过谢谢所以活下来"
- "光头厨师宣称用厨艺征服了鲱鱼罐头"

### ✅ 剧本角色一致性
- 光头厨师：全程在场，有独特行为（摸光头/做菜/做梦）
- 马斯克：SpaceX连帽衫+挑蜡烛+火星+0.01%幸存者
- 会说话的猫：高傲、叼食物、给罐头
