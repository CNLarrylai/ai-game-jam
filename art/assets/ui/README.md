# UI 套件接线说明（UI 美术产出）

所有 PNG 透明底、原始像素尺寸，游戏内用 `image-rendering: pixelated` 整数倍放大。
色板：废土 28 色白名单；UI 主用 F 组中性 + 语义强调色。**全套无 dithering、无半透明渐变、无圆角抗锯齿。**
